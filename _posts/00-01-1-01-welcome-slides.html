Command Line

Let's look at basic commands

The command line can be intimidating, that's okay.

pwd

Print the current working directory

ls [path]

List the contents of a directory (or the current directory)

cd <path>

Change the current working directory

cd ~

Go to your home directory

cd ..

The .. indicates the parent directory

cp <src> <dest>
mv <src> <dest>
rm <path>

Copy, move, and remove files

man <command>

Get “help”

Man pages are often very difficult to understand.

How do I …?

Ask.

You can do just about anything on the command line.

Learning how to use the command line will make you more productive.
